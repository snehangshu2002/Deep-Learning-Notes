Customer Churn Prediction System
Machine Learning · Classification

Build an end-to-end ML pipeline that predicts which customers are likely to cancel a subscription. Use a real Kaggle telecom/bank churn dataset, perform feature engineering, train multiple classifiers (Logistic Regression, XGBoost, Random Forest), and compare performance. Deploy a simple Flask/Streamlit dashboard showing churn probability per customer.

📄 Copy-Paste Resume Bullets

  • Implemented full ML pipeline (preprocessing → SMOTE oversampling → hyperparameter tuning via Optuna) reducing false-negative rate by 18%.
  • Deployed interactive Streamlit dashboard for real-time churn probability scoring, reducing manual analysis time by ~4 hrs/week.

NLP Sentiment Analysis & Topic Modeling
NLP · Unsupervised Learning

📄 Copy-Paste Resume Bullets
  • Processed 50,000+ Amazon reviews using BERT-based sentiment classifier (fine-tuned on domain data), achieving 89% F1-score vs. 74% for VADER baseline.
  • Applied BERTopic to extract 12 coherent product complaint themes, surfacing actionable insights adopted by product team in roadmap planning.
  • Built Plotly Dash visualization tracking sentiment shift over 18 months, revealing 34% negativity spike correlated with a firmware update.

House Price Prediction with Explainability
Regression · Explainable AI (XAI)

Go beyond basic linear regression on the Ames Housing dataset. Engineer advanced geospatial & interaction features, train a stacked ensemble (Ridge + Gradient Boosting), and — critically — use SHAP values to explain predictions. This XAI angle is extremely sought-after in data science roles.

📄 Copy-Paste Resume Bullets
  • Built a stacked regression ensemble (Ridge + LightGBM) on 80+ engineered features, achieving RMSE of $14,200 (top 8% on Kaggle leaderboard).
  • Applied SHAP (SHapley Additive exPlanations) to quantify feature contributions per prediction, producing interpretable reports for non-technical stakeholders.
  • Automated data cleaning pipeline handling 20+ missing-value patterns and skewed distributions via Yeo-Johnson transforms, reducing preprocessing time by 60%.

  • Delivered an interactive Tableau dashboard tracking MoM sales trends and product performance, used to inform $200K inventory reallocation decision.

Time Series Demand Forecasting
Time Series · Forecasting

Forecast retail or energy demand using classical (SARIMA, Holt-Winters) and modern (Prophet, LSTM) methods. Compare models with MAPE/RMSE. Add seasonality decomposition and anomaly detection. Deploy as a REST API with FastAPI. This is extremely relevant for supply chain, finance, and ops roles.

📄 Copy-Paste Resume Bullets
  • Developed multi-model demand forecasting system (SARIMA, Facebook Prophet, LSTM) on 3 years of retail data, with Prophet achieving MAPE of 6.2%, outperforming business baseline by 41%.
  • Implemented STL decomposition and isolation forest anomaly detection identifying 23 demand spikes linked to promotional events.
  • Packaged forecast models into FastAPI REST endpoint with automated weekly retraining, cutting manual forecasting effort by 8 hrs/week.
